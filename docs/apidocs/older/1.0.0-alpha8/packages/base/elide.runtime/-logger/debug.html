expect open fun debug(vararg message: Any)

Log one or more arbitrary messages to the console or log, at the level of LogLevel.DEBUG.

Each argument is expected to be a string. For automatic string conversion or direct log level control, see log. To engage in string formatting, or callable log messages, see other variants of this same method.

expect open fun debug(producer: () -> String)

Log the message produced by the provided producer, at the level of LogLevel.DEBUG, assuming debug-level logging is currently enabled.

If debug logging is not active, the producer will not be dispatched.

producer

Function that produces the message to log.
