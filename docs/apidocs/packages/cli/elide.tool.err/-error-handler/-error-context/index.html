Error Context

Describes context surrounding an error event which is being reported to the error handler. This record carries such details as the stacktrace, top error coordinates, and the active language runtime.

Parameters

uuid

UUID for this error; transient for the purposes of serialization.

fatal

Whether the error was fatal and caused a crash; defaults to true.

guest

Indicates that this error arose during execution of guest code.

coordinates

Indicates where the error took place (line, file, etc.), if known.

timestamp

Timestamp indicating when this error arose.

stacktrace

Stacktrace for the error, if one has already been generated.

frames

Frames indicating where this error took place, if already known.

language

Language runtime which was active when this error arose.

thread

Thread where this error occurred; transient for the purposes of serialization.
