ErrorStackFrame

@JvmRecord
@Serializable
data class ErrorStackFrame : Record

Error: Stack Frame

Describes a single stack frame within the stack trace for an error; this record is serializable, allowing a set of frames to be written to disk.

Parameters

moduleName

Name of the JVM module from which this error arose, as applicable.

moduleVersion

Version of the JVM module from which this error arose, as applicable.

classLoaderName

Class-loader which loaded the class where this error took place, as applicable.

className

Name of the class where this error took place, as applicable.

methodName

Name of the method or function where this error took place, as applicable.

fileName

Name of the file where this error took place, as applicable.

lineNumber

Source line number where this error took place, as applicable.

columnNumber

Source line column number where this error took place, as applicable.

isNativeMethod

Indicates whether this frame is describing a native method (if not, it is a JVM method).
