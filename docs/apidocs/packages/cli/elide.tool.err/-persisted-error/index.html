Persisted Error

Describes a single persisted error object, which was written by the DefaultStructuredErrorRecorder, and may be loaded and reported to GitHub at a later time.

Parameters

Format version; defaults to 1.

id

ID of the error record.

Timestamp indicating when the error occurred.

runtime

Info about the Elide runtime which produced this error.

args

Command-line arguments which were active when this error was produced, as applicable.

error

Information about the error itself -- the stacktrace, exception type, and so on.
