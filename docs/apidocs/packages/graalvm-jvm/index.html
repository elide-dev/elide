graalvm-jvm

elide-graalvm-jvm integrates Elide and Espresso, a high-performance Java runtime based on the Truffle framework.

How Elide uses Espresso

Elide uses Espresso to support JVM languages like Java, Kotlin, and Groovy. The Java plugin, implemented by this module, depends on the elide-jvm package.

Installation

This package is made available via Maven Central and Elide's package repository, but is not designed for end-user consumption. Language support is built-in to the Elide runtime.

Via Gradle (Catalog):

implementation(framework.elide.graalvm.jvm)
