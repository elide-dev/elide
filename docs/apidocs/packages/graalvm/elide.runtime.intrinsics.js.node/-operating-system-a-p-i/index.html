OperatingSystemAPI

@API
interface OperatingSystemAPI : NodeAPI

Node API: os

Describes the API provided by the Node API built-in os module, which supplies operating system-related utilities, such as load averages, OS constants, resource usage, and more. Access to these methods is moderated by the active Elide security policy specified by the user.

 

Summary

The default implementation of the os module supplies cross-platform logic, which works on POSIX and Win32-style systems. Operating system APIs are called on-demand. Where access is not granted to such APIs, mock or stubbed values are supplied, or errors are thrown, as applicable.
