interface ProtoBufSchemaBuilder

Protobuf: Schema Generator

Describes the interface which protocol buffer schema generators adhere to. There is only one method at the time of this writing; generateSchemaText, which generates proto-schema syntax for a given set of SerialDescriptor instances, each for a Kotlin model.
