RequestContextFilter

@Filter(value = ["/**"])
class RequestContextFilter : HttpServerFilter

Provides an HttpServerFilter which affixes context values at known keys in HttpRequests processed by Elide apps; known keys are defined via RequestContext.Key.

See also

RequestContext.Key

for an exhaustive review of available request context.
