HTTP Headers: Factory

Describes static factory methods which create immutable HTTP header mappings; see HttpHeaders for more info about how headers are stored and accessed.

Mutable HTTP header containers are accessible through the corresponding factory interface on that type, at MutableHttpHeaders.Factory. That factory extends this one.

See also

MutableHttpHeaders.Factory

to create mutable HTTP headers.
