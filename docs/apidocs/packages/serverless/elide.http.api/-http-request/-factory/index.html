interface Factory

HTTP Request: Factory

Describes the API provided for creating new HTTP requests; this is typically used to lay out the static method structure on a HTTP request implementation.

 

Mutability

Implementations of this interface are expected to be immutable, and should not allow modification of the request method, path, or query parameters after creation (or any other properties). For a mutable form of this interface, see MutableHttpRequest.Factory.

Request Behavior

See the main HttpRequest interface for more information about how HTTP requests behave under the hood.
