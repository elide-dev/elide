@API
interface HttpResponse : HttpMessage

HTTP: Response

Describes the basic API provided for HTTP responses within Elide; this extends HttpMessage with information which is specific to responses.

Features

TBD.

Response Properties

Status (status)

Describes the HTTP status for the response expressed by this record; response statuses carry a status code, and an optional text "reason" or description.

Trailers (trailers)

Describes the HTTP trailers for the response, if any. Trailers are a set of headers which are sent after the response body, and are used to carry metadata about the response. Trailers are not supported by all clients or servers, and only at HTTP/2 or later.
