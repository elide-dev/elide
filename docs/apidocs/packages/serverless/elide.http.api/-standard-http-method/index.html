HTTP: Standard Method

Enumerates standard HTTP request methods, binding them to unique identifiers for serialization, and providing various utility methods for working with HTTP methods.

Whether this operation is expected to modify the state of the server or its resources; if true, the operation is considered a write operation, and is not expected to be idempotent by default.
