class HttpHeaders : HttpHeaders, Map<HttpHeaders.HeaderName, HttpHeaders.HeaderValue>

HTTP Headers

Immutable container which maps HTTP headers to their values; supports case-insensitive and ordered comparison by header key, and preserves insertion order for header values. See HttpHeadersAPI for more information.
