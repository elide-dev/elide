## Installation

<a id="installation"></a>

Elide is supported on Linux and macOS (amd64 and arm64).

You can use our one-liner:
```
curl -sSL --tlsv1.2 elide.sh | bash -s -
```

Or Brew:
```
brew tap elide-dev/elide && brew install elide
```
